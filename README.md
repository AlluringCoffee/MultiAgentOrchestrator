# Multi-Agent Orchestrator

A powerful visual workflow designer and execution engine for multi-agent LLM systems.

## Features

- **Visual Node Editor**: Drag-and-drop interface for building complex workflows.
- **Multi-LLM Support**: Built-in support for Ollama, Groq, and Google AI.
- **WebSocket Integration**: Real-time status updates and execution logs.
- **God Mode Tools**: Integrated CLI, Git, and HuggingFace tools for agents.
- **Workflow Export & Deployment**: Export workflows to standalone scripts or Docker containers.

## Getting Started

1. Install dependencies: `pip install -r requirements.txt` and `npm install`
2. Configure environment: Copy `.env.example` to `.env` and add your API keys.
3. Run the server: `python server.py`
4. Open `http://localhost:8000` in your browser.
